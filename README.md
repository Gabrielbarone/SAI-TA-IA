# SAI TA | IA — GitHub Pages (v7)

Este pacote está pronto para publicação no GitHub Pages.

## Arquivos

- `index.html` — página principal do site.
- `.nojekyll` — garante a publicação como site estático.
- `404.html` — página para links inexistentes.

## Como atualizar o repositório

1. Abra o repositório `SAI-TA-IA` no GitHub.
2. Vá em **Code → Add file → Upload files**.
3. Envie os arquivos desta pasta diretamente na raiz do repositório e substitua o `index.html` existente.
4. Clique em **Commit changes**.
5. Em **Settings → Pages**, mantenha:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
6. Deixe **Custom domain** vazio. Caso apareça `saiiata`, use **Remove**.
7. Aguarde a publicação e acesse:
   `https://gabrielbarone.github.io/SAI-TA-IA/`

## Privacidade e publicação

`noindex` reduz a indexação em buscadores, mas não torna a página nem o repositório privados. Quem tiver a URL, ou acessar o GitHub, poderá ver o conteúdo e os links presentes no código. Não publique dados confidenciais, tokens, documentos internos ou links que não possam ser expostos.
