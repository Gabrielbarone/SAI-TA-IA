# SAI TA | IA — GitHub Pages

Este pacote já está pronto para ser publicado no GitHub Pages.

## Arquivo obrigatório

- `index.html` — página principal do site.
- `.nojekyll` — garante que o GitHub Pages trate o projeto como site estático.
- `404.html` — página exibida para links inexistentes.

## Como publicar

1. Abra o repositório `SAI-TA-IA` no GitHub.
2. Na aba **Code**, clique em **Add file → Upload files**.
3. Envie todos os arquivos desta pasta, mantendo-os na raiz do repositório.
4. Clique em **Commit changes**.
5. Acesse **Settings → Pages** e confirme:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
6. Não preencha **Custom domain**. Caso apareça `saiiata`, clique em **Remove**.
7. Aguarde o deploy e acesse:
   `https://gabrielbarone.github.io/SAI-TA-IA/`

## Privacidade e indexação

O `index.html` inclui `noindex, nofollow, noarchive`, instruindo buscadores a não indexar o site. Ainda assim, o repositório e a URL continuam públicos e acessíveis por link direto. Não suba credenciais, tokens, informações confidenciais ou documentos internos.
